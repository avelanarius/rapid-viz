Opis szczegółowy

Klasa opisująca dostępne style linii (grot początkowy i końcowy).

Dostępne są 4 typy grotów:

  • BRAK
  • KOLO
  • TROJKAT
  • KWADRAT

Każdy styl opisany jest w następującym formacie: STYL_[Grot początkowy]_[Grot końcowy].
