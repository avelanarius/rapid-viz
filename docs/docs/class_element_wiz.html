Opis szczegółowy

Klasa opisuje obiekt widoczny w oknie wizualizacji, taki jak kształt lub linia.

Dostępne są trzy typy ElementWiz:

  • ksztalt: jedna z figur geometrycznych dostępnych w klasie Ksztalt (m.in. kwadrat, koło, trójkąt, drzewo)
  • linia: linia pomiędzy dwoma punktami, z możliwością ustawienia grotów (StylLinii)
  • liczba: całkowita liczba wypełniająca zadany prostokąt.
