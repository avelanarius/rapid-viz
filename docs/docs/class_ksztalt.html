Opis szczegółowy

Klasa opisująca dostępne kształty, takie jak KOLO, KWADRAT.

Poszczególne typy dostępne są jako właśność statyczna, przykładowo: Ksztalt::KOLO, Ksztalt::KWADRAT. Istnieje kształt Ksztalt::BRAK określający nieprawidłowy kształt, brak kształtu.
